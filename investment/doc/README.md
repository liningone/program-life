# 投资

## 需求说明
1.用股市历史数据验证定投策略的收益情况    &nbsp;  
2.支持指定目标股票/指数，时间长度，定投周期，定投间隔，定投金额，止盈线，止损线，定投条件  &nbsp;  
3.给出收益率情况，包括期间最大的亏损收益，最大的收益率，收益率曲线，亏损的天数    &nbsp;  
4.(未来需求) 定投策略横向评分（可以基于风险偏好）     &nbsp;  
5.(未来需求) 根据股票历史数据挖掘出最佳的定投策略     &nbsp;
